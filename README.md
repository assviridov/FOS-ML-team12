# Фонд оценочных средств по дисциплине «Машинное обучение»

Этот репозиторий предназначен для открытого и воспроизводимого представления фонда оценочных средств (ФОС) по дисциплине «Машинное обучение». Здесь объединены рабочая программа дисциплины, модель измерения результатов обучения, контрольно-измерительные материалы (КИМ), критерии оценивания, методические рекомендации и используемые образовательные ресурсы. Результаты обучения связываются с компетентностно-ролевой моделью в области искусственного интеллекта (КРМ версии 3.0).

## 1. О дисциплине

**Цель дисциплины.** 
Целью освоения дисциплины является формирование у обучающихся фундаментальных знаний в области машинного обучения, включая базовые алгоритмы, методы обработки и анализа данных, а также развитие навыков практического применения этих методов при решении инженерных и научных задач.


**Основные задачи.** 
- изучить основные теоретические основы алгоритмов обучения с учителем и без учителя;
- освоить методы подготовки данных, оценки качества моделей и выбора алгоритмов;
- приобрести навыки практического применения алгоритмов машинного обучения с использованием современных языков и библиотек программирования;
- развить умения анализа данных и обоснованного выбора моделей для решения прикладных задач;
- подготовить студентов к курсовому проектированию, исследовательской деятельности и выполнению выпускной квалификационной работы в области анализа данных и интеллектуальных систем.

**Структура и содержание.** Дисциплина состоит из трех модулей: Предобработка данных, Основы классического машинного обучения,
Машинное обучение на Python и R. Обучение включает лекции / лабораторные работы / проект / самостоятельную работу] и завершается зачетом.

**Пререквизиты и связи с другими дисциплинами.** 
Дисциплина «Машинное обучение» относится к вариативной части Блока 1 «Дисциплины (модули)» учебного плана. Данная дисциплина («Машинное обучение») тесно связана с дисциплинами: 

предикаты 
- Математический анализ (MF-1),
- Векторная алгебра (MF-1),
- Курс теории вероятностей( MF-1, MF-2) , 
- Базы данных (BD-3)
- Обработка данных  (BD-1, BD-2)
  
А также последующие:
- Глубокое обучение (DL-1, DL-2),
- Нейросетевые технологии (DL-1, ML-4),
- Безопасность информационных систем (ML-5),
- Гибридный ИИ: математическое моделирование и МО, проектировании (MF-3 П) 
- Выполнении выпускной квалификационной работы (ML-2 П, ML-3 П, MF-7)
 
**Планируемые результаты обучения.** По завершении дисциплины обучающийся сможет:
- Применять методы предварительной обработки данных (обработка пропусков, выбросов, масштабирование, кодирование категориальных признаков) и создавать воспроизводимые пайплайны преобразований с использованием библиотек Python (Scikit-learn Pipeline, ColumnTransformer) для подготовки данных к обучению моделей.
- Обосновывать выбор конкретных алгоритмов машинного обучения (линейные модели, k‑NN, SVM, деревья решений, ансамбли) и их гиперпараметров в зависимости от типа задачи (регрессия, бинарная/многоклассовая классификация), размера и структуры данных, а также требований к интерпретируемости и вычислительной эффективности.
- Реализовывать полный цикл обучения моделей с учителем: разделение выборки, кросс-валидацию, подбор гиперпараметров (GridSearchCV / RandomizedSearchCV), оценку качества с использованием адекватных метрик (MAE, RMSE, R², Accuracy, Precision, Recall, F1, ROC‑AUC) и диагностику недообучения/переобучения.
- Применять методы ансамблирования (бэггинг, случайный лес, градиентный бустинг) и классические подходы к анализу временных рядов (линейная регрессия с лагами, экспоненциальное сглаживание) для повышения точности прогнозов и устойчивости моделей.
- Использовать алгоритмы обучения без учителя (кластеризация K‑means, DBSCAN, понижение размерности PCA, обнаружение аномалий Isolation Forest / One‑Class SVM, ассоциативные правила Apriori) для выявления скрытых закономерностей, структурирования данных и визуализации многомерных пространств.
- Оценивать этические и правовые аспекты работы с цифровым следом: проводить аудит наборов данных на предмет скрытых смещений, репрезентативности выборки и законности источников данных, а также интерпретировать результаты моделей с учётом ограничений и возможных социальных последствий.

Полное описание приведено в [РПД](docs/rpd.md).
