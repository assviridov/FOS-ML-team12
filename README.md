# Фонд оценочных средств по дисциплине «Машинное обучение»

> **Статус:** шаблон репозитория. Замените все маркеры `[ЗАПОЛНИТЬ]` и удалите учебные примеры перед публикацией.

Этот репозиторий предназначен для открытого и воспроизводимого представления фонда оценочных средств (ФОС) по дисциплине «[Машинное обучение]». Здесь объединены рабочая программа дисциплины, модель измерения результатов обучения, контрольно-измерительные материалы (КИМ), критерии оценивания, методические рекомендации и используемые образовательные ресурсы. Результаты обучения связываются с компетентностно-ролевой моделью в области искусственного интеллекта (КРМ версии 3.0).

## 1. О дисциплине

**Цель дисциплины.** 
Цели изучения дисциплины определены государственным образовательным стандартом высшего образования и соотнесены с общими целями ООП ВО по направлению подготовки 09.03.03 Прикладная информатика, в рамках которой преподается дисциплина.
Цели дисциплины Машинное обучение:
-	познакомить студентов с основными разделами искусственного интеллекта;
-	научить студентов правильно выбирать методы решения задач ИИ в соответствии с поставленной задачей;
-	научить студентов проводить предварительный анализ данных и подготовку данных для дальнейшего использования в задачах машинного обучения с помощью языков R и Python.


**Основные задачи.** 
-	изучить базовые понятия машинного обучения;
-	познакомить студентов с основными этапами анализа и подготовки обучающих данных;
-	изучить основные алгоритмы классического машинного обучения;
-	изучить библиотеки, необходимые при работе с классическим машинным обучением на Python (Seaborn, Scikit learn,  Matplotlib, Pandas) и с помощью языка R.

**Структура и содержание.** Дисциплина состоит из трех модулей: Предобработка данных, Основы классического машинного обучения,
Машинное обучение на Python и R. Обучение включает лекции / лабораторные работы / проект / самостоятельную работу] и завершается зачетом.

**Пререквизиты и связи с другими дисциплинами.** 
Данная дисциплина («Машинное обучение») тесно связана с дисциплинами: Математический анализ, Векторная алгебра, Курс теории вероятностей, Математические модели нейронных сетей,  а также при курсовом проектировании и выполнении выпускной квалификационной работы.

**Планируемые результаты обучения.** По завершении дисциплины обучающийся сможет [ЗАПОЛНИТЬ: 3–6 проверяемых результатов обучения, сформулированных через наблюдаемые действия].

Полное описание приведено в [РПД](docs/rpd.md).

## 7. Порядок заполнения

1. Заполните РПД и сформулируйте проверяемые результаты обучения.
2. Выберите релевантные профессиональные роли и компетенции КРМ.
3. Для каждого результата выберите индикатор и требуемый уровень.
4. Заполните модель измерения.
5. Создайте КИМ и рубрики оценивания по шаблонам.
6. Согласуйте систему итогового балла.
7. Добавьте ресурсы, сведения о команде и лицензию.
