# Модуль 1. Основы ML-проекта, данные и оценка качества

Модуль включает три учебных практикума. Студент осваивает структуру ML-задачи, первичный анализ данных, preprocessing и выбор метрик. Модуль завершается индивидуальной самостоятельной работой № 1.

| Параметр | Значение |
|----------|----------|
| **Учебные практикумы** | УП 1–3 |
| **Компетенция** | ML-2 |
| **Индикаторы** | ML-2.1; ML-2.2; ML-2.3 |
| **Самостоятельная работа** | СР № 1 |
| **Вес СР** | 12,5% |
